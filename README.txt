##Movie Website:

This is a code that creates an html page with my favourite movies, displays their poster images and plays movie's trailer when click on it's image.

##How to open the web page:

**Python and Python IDLE** should be installed on your computer first. Open entertainment_center.py file in python IDLE and select run module from the run 
drop down lis or press F5. A web page will be opened with the movies on it. press on any movie image to play the movie trailer.
